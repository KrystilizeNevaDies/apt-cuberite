## apt-cuberite
A plugin package manager for cuberite

## Usage

# apt help
(Displays help menu)

E.G.
apt help

# apt info
(Displays info on plugin)

E.G.
apt info realtime


# apt install
(Installs plugin and dependencies)

E.G.
apt install worldedit


# apt remove
(removes plugin)

E.G.
apt remove worldedit
