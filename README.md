# apt-cuberite
A plugin package manager for cuberite

# Usage


## apt help
(Displays help menu)

E.G.
apt help


## apt install
(Installs plugin)

E.G.
apt install core portals

## apt installall
(Installs all available plugins)

E.G.
apt installall


## apt remove
(removes plugin)

E.G.
apt remove essentials coreanticheat

## apt removeall
(remove plugins)

E.G.
apt removeall


## apt update
(updates a plugin)

E.G.
apt update shopify particletrails

## apt updateall
(update plugins)

E.G.
apt updateall


## apt info
(Displays info on plugin)

E.G.
apt info bankapi


## apt list
(Lists all installed apt plugins)

E.G.
apt list


## apt list all
(lists all available plugins)

E.G.
apt list all
